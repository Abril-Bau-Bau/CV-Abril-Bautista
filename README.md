# CV | Abril Bautista Gómez

Currículum web personal desarrollado como proyecto de Diseño Avanzado.

## Contenido

- Perfil profesional y experiencia laboral
- Proyectos relevantes
- Habilidades y herramientas
- Formación académica y certificaciones
- Enlaces de contacto y portafolio

## Tecnologías

- HTML5 semántico
- CSS3 personalizado
- Bootstrap 5
- JavaScript
- Diseño responsive mobile first

## Funciones

La página incluye navegación adaptable, modo oscuro, impresión del CV, botón para volver al inicio y estilos accesibles para teclado y movimiento reducido.
